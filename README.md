# 🍔 FoodHub - Online Food Ordering Website

FoodHub is a modern and responsive **food ordering website frontend project** built using **HTML, CSS, and JavaScript**. The website provides a Zomato-style user experience where users can explore different food categories, view popular dishes, add items to the cart, and interact with a login system.

This project is developed as a **college frontend project** to demonstrate the use of web development technologies, responsive design, DOM manipulation, and JavaScript-based cart functionality.

---

## 🚀 Features

### 🏠 Home Page
- Attractive hero section with food background image
- Search bar for searching food items
- Explore food categories
- Popular dishes section
- Responsive navigation bar

### 🥗 Veg Food Section
- Dedicated vegetarian food menu
- Categories:
  - Fast Food
  - Main Course
  - South Indian
  - Desserts
- Add food items to cart functionality

### 🍗 Non-Veg Food Section
- Dedicated non-vegetarian food menu
- Categories:
  - Chicken Specials
  - Mutton Specials
  - Seafood
  - Non-Veg Fast Food
- Interactive food cards

### 🛒 Shopping Cart
- Add items to cart using JavaScript
- Cart sidebar with smooth animation
- Increase/decrease item quantity
- Remove items from cart
- Automatic price calculation
- Cart data stored using Session Storage

### 🔐 Login System
- Login popup modal
- User input fields:
  - Username
  - Phone Number
  - Email
  - Password
- Form validation using HTML required attributes

### 📞 Contact Page
- Contact information section
- Message sending form
- Social media icons

### 🔍 Search Feature
- Search food items dynamically
- Filters food cards according to user input
